# Grupo de Dados (GD)

Este repositório hospeda os projetos desenvolvidos e mantidos pelo Grupo de Dados (GD) da Divisão de Previsão de Tempo e Clima (DIPTC) do Centro de Previsão de Tempo e Estudos Climáticos (CPTEC).


### Organização do Grupo de Dados

O GD é divido em sub-grupos:

> Aquisição de Dados
> Pré-processamento
> Banco de Dados

### Organização de Projetos

O GitHub é utilizado para o gerenciamento e documentação dos projetos em desenvolvimento pelo Grupo de Dados. Cada novo projeto do GD deve possuir um repositórios específico.
